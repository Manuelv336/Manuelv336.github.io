Lab 02 Reflection: HTML

Learning HTML basics is fun and I think I've got the hang of it. Playing with the tags felt like building a pyramid: I started with the foundation html tag, moved up to the head and body, and then the rest of the levels, such as the header, main, and footer. The body is the main part. I will use the Google Fonts website in the list to see what fonts I’ll add to my future websites, and I use the playground as a sandbox to quickly see how a website looks when I'm testing my code. W3Schools is my go-to website to research different looks and options of html tags. It all feels pretty easy, almost like matching pieces in a puzzle. I treat every project like a challenge, and it's nice to look at a small html files now and read what’s going on. Im sure i would struggle with a large file. Indentation is still a bit of a struggle for me, but I'm getting there. I’m a curious person, and I’m really into learning new skills. Building websites has been on my bucket list for a while, and I’m getting pretty excited to keep leveling this project up. I'm looking forward to learn how to add tabs, js and incorporate some Excel features.

Lab 03 Reflection: External CSS

I enjoyed completing this assignment. It gives character to the HTML; most of the CSS came from my reading notes, while others, like the padding and line-height, came from experimenting in Visual Studio Code. For the body, I like soft colors like blue and gray. For the headings, I chose navy blue so they stand out more, and I added spacing to the paragraphs to make them easier to read. For the links, it is cool how you can add an effect to change the color when you hover over them; I'll use this on my next project when I add clickable boxes. The ID selector was a challenge for me, but I eventually realized it wasn't connecting to the HTML because of a capital letter. Now I know that CSS is case-sensitive! I was also overlapping styles by adding alignment to both the body and the paragraphs, so I removed some of the redundant CSS. I feel like there is an overwhelming amount of styles out there. I used padding on the body, but I’m not sure if it’s better to use pixels or percentages. Do you know that Visual Studio auto-completes the coding when you start typing?
