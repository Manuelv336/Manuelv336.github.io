The body is the main part. W3Schools is my go-to website to research different looks and options of html tags. It all feels pretty easy, almost like matching pieces in a puzzle. I treat every project like a challenge, and it's nice to look at a small html files now and read what’s going on. Im sure i would struggle with a large file. Indentation is still a bit of a struggle for me, but I'm getting there. I’m a curious person, and I’m really into learning new skills. Building websites has been on my bucket list for a while, and I’m getting pretty excited to keep leveling this project up. Lab 03 Reflection: External CSS

Now that I’ve got the html basics down, it’s time to decorate that pyramid with CSS. This is the part I was looking forward to and it feels a lot like decorating, mapping styles from one file to the other to see how things shift si fun. I think this is what gives a website its character; I can make it friendly, professional, or even look like a rainbow. After reading the material, I’ve been asking myself these three questions for every change: Where do I put the code? Should I make these changes inline, internal, or external? What am I targeting? Am I styling the title, the body, or a specific line? and what changes am I making? Do I need to adjust the color, font, or alignment? In this case, for lab 3, I was given the what and how and used my creativity to make the changes. I found it easy to complete but a bit tricky when doing it external then internal. At first, I did strugle on adding the CSS from a external file but I went back on the reading and find out how to refence the file if its in the same forlder.
