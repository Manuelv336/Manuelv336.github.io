My presonal reflection for class CSIT 121

This was a well structured class, and I was able to learn so much as a beginner in web development. I can also say that I can place a check mark on making a website. All the resources available from the class content make it easier for me to complete this course. This is the first course I have taken where I do not have an instructor guiding me on how to complete each step, and I can say that it was not necessary because it was well organized by the professor, who made great videos demonstrating what needed to be done, and even though I didn't need much help, I knew that she was available if I ever needed help.

One of my struggles in this class was with the writing of HTML, adding capital letters and remembering the correct elements. But I was able to get through and fix my mistakes by reviewing multiple times until I felt it's complete. I know practice makes perfect, and I will not stop here. Another thing I struggled with was the spacing to tell which one is the parent and child. I also struggled with using media queries, so I had to go back and find out how to add the screen type to my CSS. I'm sure with more practice I'll get the hang of it. My plan is to continue to grow in this field and develop for fun.

I'm using Lab 10 as the base of this project and elaborating from there. Challenges I face were time, because it's the end of the semester, I had to study for the final and other project from my other classes. I successfully integrated a table, an image, and a list. One thing that is not required is the access back to this website from the individual labs or having the labs open on a separate web without closing the current page, I know it can be done with Ctrl + click, but I wonder if there is an automated way. I corrected the issue of orphaned pages by adding the target="_blank" attribute to all my links. This ensures that labs and all links open in a new window or tab, allowing the user to easily return to my main website.

I was able to successfully able to test my website on Chrome, Brave and Safari and did not encounter any issues. I decided to keep my Nav bar vertical so that it can display correctly on mobile and computer.
