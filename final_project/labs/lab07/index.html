Lab 07 Reflection

This is my reflection on the lab experience.

A challenge I encountered:
This lab was a big challenge because it felt like it combined everything we have learned so far into one single project. My biggest mistake at the start was trying to work line-by-line through the requests, which made it hard because I had to go back and update each element and make sure the colors matching and my layout was organized. I had a lot of issues with the images not loading or being way too big, and I struggled with getting the alignment and spacing right between the different sections. I was also a bit confused about how many HTML files I actually needed. I thought I only needed two, but after checking the rubric, I realized I needed three. I was not able to find the java scrip I needed to modify.
How I worked through it:
To work through these issues, I started by thinking about a theme, which for me was food, and I built a basic shell of the requirements of 6 x 6 before adding any CSS. When I ran into the issue of images being too large, I fixed it by reducing the image sizes, for some reason when I test my code in visual studio it does not show until I make it smaller. To make the image small enough to fit the table I added width and height style. Because I was not able to find the java scrip I search in W3Schools and use it as a resource to help with the JavaScript. I'm familiar with scrips and does a check to see if the name or email was added was easy to me. once the shell for the html was done I added all the styles in one file to all three htmls and try to keep all the alignment consistence. I like the hover option I added to the navigation.
Something that clicked:
Something that clicked for me during this project was how important file naming, tag taming and project structure are. I know from previous labs that the home page usually starts as index.html, but I got mixed up with the specific naming in the instructions at first. I eventually realized I needed to use index and use it was the home page so that everything would link together correctly.
A question or concept you are still unsure about:
I am still a little unsure about a few concepts, like if there is a way to make images clickable so that they can expand for the user. I also wonder if I handled the header colors the best way possible. When I was trying to use the odd and even row colors, the header kept getting caught up in that styling, but I wanted it to have its own unique color. I managed to make it work, but I’m curious if there’s a better way to target the header so it stays consistent without the other CSS rules overwriting it.
