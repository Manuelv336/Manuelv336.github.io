The technical foundation of this project primarily used documentation from MDN Web Docs and web.dev. MDN provided the main specifications for KeyboardEvent.code, which helped me understand how it is currently being used and allowed me to practice the "play" functionality myself.

To provide a clear example, I implemented a live demonstration. I initially copied the code from MDN Web Docs and pasted it into my HTML, applying CSS to match the style of my website. When the code didn't work as expected, I used ChatGPT to help troubleshoot and fix the implementation.

To improve the UI and visual representation, I used a picture found on Rawpixel that is free of copyrights. This gives the website a much better visual effect.
