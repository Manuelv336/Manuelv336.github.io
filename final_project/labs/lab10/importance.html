Why is UI Events: Keyboard Codes Important?

UI Events: Keyboard Codes is important because standard keyboards have limitations on the amount of keys available. Implementing more key functions makes it easy and gives more functionality to the website. Implementing new keys can actually increase the user experience on the web page. I can see this saving a lot of time by creating shortcuts such as print with the press of a button.

A second way I can see the benefit and importance of implementing UI events: Keyboard is the placement and layout of key functions. I know when I play video games on a PC it is important to use both hands at the same time. On a standard gaming controller, they place the buttons in a way that they are easily pressed by a specific finger, and this standard could do the same.

Lastly, this standard gives the user control. If combined with JavaScript, it can give the website visibility and functionality not available on standard websites. Most websites just think about what you see and are based on one click. Adding codes to the website gives that control for the user to be able to access more functionalities.
